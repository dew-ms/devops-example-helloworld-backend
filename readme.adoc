==== DevOps示例 : Helloworld后端服务

本示例为Dew微服务体系的组成部分，使用说明参见：http://doc.dew.ms

.说明
此示例用于演示后端服务的自动化部署到Kubernetes，完成后推送钉钉通知。

[source,yml]
./.dew配置说明
----
include::.dew[]
----

.前置准备

. 执行 ``dew-devops.sh`` 初始化项目对应的test环境（ 详见 http://doc.dew.ms/#devops-user-manual ）

.手工执行
----
# 参数设置
SET dew.devops.kube.config=...
SET dew.devops.docker.host=...
SET dew.devops.docker.registry.url=...
SET dew.devops.docker.registry.username=...
SET dew.devops.docker.registry.password=...

# 执行如下maven命令(发布到test环境)
mvn -P devops dew:release -Ddew.devops.profile=test
----
